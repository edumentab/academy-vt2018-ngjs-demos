AngularJS does dependency injection by looking at parameter names. So for example when we have a parameter named exactly $scope, we get that service.

But minification often renames parameters, which would break everything! AngularJS therefore has an alternate syntax where we name the parameters in an array, and have the actual function as the last element.

{{message}}
index.html
main.ts
<div ng-app="helloworld">
  <div ng-controller="helloworldcontroller">
    {{message}}
  </div>
</div>
import * as angular from 'angular';

let app = angular.module('helloworld', []);

// Now the parameter will be the $scope service,
// and we can call it whatever!
app.controller('helloworldcontroller', ['$scope', function(a){
  a.message = 'Hello world!';
}]);
